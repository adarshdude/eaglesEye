The Eagle's eye is the project based on the surveillance system detection of the object that the ip cam records, this recording can be imported from the number of input devices, even your mobile camera. The code works on the detection of the motion of the object in the frame and then they are snapped and saved in the system memory which can further be used for security purposes.It is the high end security system which runs in the python's OpenCV library.

In addition to the simple streaming camera we have provided it with a system to serve the best security purposes. The simple webcam just provides the video recoring with the static and dynamic objects and therefore needs full monitoring and the person to check it and on the other hand our project provides the user to sit back and relax while the cam will do all the security work and will send the alert to the user in case of any dynamic motion, the snaps will be taken at the rate of 50 snaps per second. This will compare it with detection in motion, if the motion is detacted then the snaps will be saved to the system for further monitoring.
